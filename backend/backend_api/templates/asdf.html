Response
Each category object in the response array will have the following parameters:

  • "id" (integer): The ID of the category.
  • "category" (string): The name of the category.
  • "isActive" (boolean): The status of the category.
Response example 

{
    "status": "success",
    "data": [
        {
            "id": 1,
            "category": "Wireless headphones",
            "isActive": true
        },
        ...
    ]
}
