Product Endpoint:

URL: /api/v1/products/
Allowed Methods: GET

Description:
The product endpoint allows you to get all the existing products from the API.

Request Body:
None
Response Body:
The response body for GET requests to the product endpoint will be in JSON format and include an array of product objects. Each product object will have the following fields: 


    "status": String,
    "data": [{
            "id": Number,
            "name": String,
            "description": String,
            "price": Number,
            "oldPrice": Number,
            "isApple": Boolean,
            "isActive": Boolean,
            "category": String,
            "mainImage": String(url to image),
            "images": [
                {
                    String(url to image)
                }
            ],
            "meanReview": Number,
            "created_at": String
        }]
